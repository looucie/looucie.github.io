MagicBall

Dans le cadre d’un projet universitaire en BUT MMI, j’ai réalisé un site web de collection de pokemon.

L’objectif était de créer une application web affichant le contenu d'une collection de Pokemon pour un seul utilisateur.

Etapes clés
  • Conception d'une planche univers
  • Réalisation d'une maquette en version papier
  • Création d’une maquette numérique
  • Production du site en HTML / CSS / JS
